## Spring Boot with Kotlin Server-side App

### Prerequisites:
- Java 11+
- Node 12+
- Yarn
- HTTPie
- Okta Developer Account.<br>You need `yourClientId` and `yourOktaUrl`.

### Dependencies:
- Gradle
- Kotlin
- Spring MVC
- Spring JPA
- Rest Repositories
- H2 DB
- Okta Spring Boot starter

### Build and run
Execute: `./gradlew bootRun`<br>
Verify: `http :8080`
